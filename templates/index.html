Help & Instructions
How to Use the Drowning Detection System

This system uses artificial intelligence to detect drowning incidents in video footage. Follow these steps to use the system:

Step 1: Upload Video

Click the "Choose File" button to select a video file from your device. Supported formats include MP4, AVI, MOV, and WMV.

Step 2: Start Analysis

Click the "Start Analysis" button to begin processing the video. The system will analyze each frame to detect potential drowning incidents.

Step 3: Monitor Detection

Watch the video feed and monitor the detection log for alerts. If a potential drowning is detected, a red alert banner will appear with an audible warning.

Step 4: Adjust Settings (Optional)

You can adjust the confidence threshold and detection mode to fine-tune the system's sensitivity based on your specific needs.

Understanding the Interface
  • Video Feed: Displays the video with detection overlays
  • Detection Timeline: Shows drowning incidents over time
  • Metrics: Displays key statistics about the current session
  • Detection Log: Lists all detection events with timestamps
  • Settings Panel: Allows customization of detection parameters
About DrownGuard

DrownGuard Drowning Detection System

Version 1.0

DrownGuard is an advanced AI-powered drowning detection system designed to enhance safety in aquatic environments. Using computer vision and machine learning algorithms, it can identify potential drowning incidents in real-time.

Key Features:
  • Real-time drowning detection using advanced ML models
  • Multi-person tracking and analysis
  • High-confidence alerts with minimal false positives
  • Comprehensive detection reports and analytics
  • Customizable detection parameters
Technology Stack:

This system uses PyTorch for ML modeling, OpenCV for computer vision, and Flask for the web interface. The frontend is built with HTML, CSS, JavaScript, and Bootstrap.

Note: This system is designed as an additional safety measure and should not replace proper supervision by trained lifeguards or swimming instructors.
